# 📹 Video Call App

A real-time video calling web application built using **React.js** and **ZEGOCLOUD UI Kit Prebuilt**. Users can create or join video call rooms instantly by sharing a unique room link.

---

## 🚀 Features

* 🎥 Real-time video calling
* 🎤 Audio controls (mute/unmute)
* 📷 Camera controls (on/off)
* 🔗 Share room links instantly
* 👥 Join existing meeting rooms
* ⚡ Fast and responsive UI
* 🌐 Browser-based communication

---

## 🛠️ Tech Stack

* React.js
* React Router DOM
* ZEGOCLOUD UI Kit Prebuilt
* JavaScript
* HTML5

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/video-call-app.git
cd video-call-app
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```

The application will run on:

```bash
http://localhost:3000
```

## 🎯 How It Works

1. User enters their name.
2. A room ID is generated or entered.
3. A shareable meeting link is created.
4. Participants join using the same room link.
5. ZEGOCLOUD handles video/audio communication.
